## Описание
Chat - веб-мессенджер


## Установка и запуск

- `npm install` — установка зависимостей,
- `npm run dev` — запуск dev-версии для разработки,
- `npm run build` — запуск build сборки для деплоя,
- `npm run start` — сборка build версии и поднятие node.js сервера.
